# TETRIS 99

**99 人俄罗斯方块对战 · iOS 27 磨砂玻璃设计风格**

[![Play](https://img.shields.io/badge/🎮_Play_Now-Click_Here-brightgreen?style=flat-square)](https://xmgzxmgz.github.io/TETRIS-99/)

纯前端实现，零依赖，单文件自包含。

</div>

---

## 特性

- 🎮 经典俄罗斯方块 99 人对战
- 🤖 AI 对手（easy / medium / hard / expert 四种难度）
- 🎯 4 种攻击策略（随机 / 攻击者 / KO / 徽章）
- 🎨 iOS 27 磨砂玻璃 UI
- 📱 响应式布局

## 操作

| 按键 | 功能 |
|------|------|
| `← →` | 移动 |
| `↑` / `X` | 顺时针旋转 |
| `Z` | 逆时针旋转 |
| `↓` | 软降 |
| `空格` | 硬降 |
| `C` | 保留方块 |
| `1-4` | 切换攻击策略 |

## 技术栈

- HTML5 + CSS3 + JavaScript
- Canvas 渲染
- 零外部依赖

## 项目结构

```
TETRIS-99/
├── index.html       # 页面
├── styles.css       # iOS 27 风格样式
├── js/
│   └── game.js      # 全部游戏逻辑
├── favicon.svg
└── README.md
```

## 本地运行

```bash
git clone https://github.com/xmgzxmgz/TETRIS-99.git
cd TETRIS-99
open index.html
```
